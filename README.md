# openlist-style

#### 介绍
自定义openlist样式

#### 使用

##### 自定义头部
```html
<!-- 1 -->
```



##### 自定义内容

```html
<!-- 图片壁纸(需要修改图片地址) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/iyuwei/openlist-style@main/css/bg_img.css?features=String.prototype.replaceAll">
```
```html
<!-- 透明组件 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/iyuwei/openlist-style@main/css/opcity.css?features=String.prototype.replaceAll">
```



```html
<!-- 视频壁纸 -->
<script type="module" src="https://cdn.jsdelivr.net/gh/iyuwei/openlist-style@main/js/bg_video.js?features=String.prototype.replaceAll"></script>
```
```html
<!-- 点击心形 -->
<script type="module" src="https://cdn.jsdelivr.net/gh/iyuwei/openlist-style@main/js/mouse_click.js?features=String.prototype.replaceAll"></script>
```
```html
<!-- 删除footer -->
<script type="module" src="https://cdn.jsdelivr.net/gh/iyuwei/openlist-style@main/js/delete_footer.js?features=String.prototype.replaceAll"></script>
```
